capstoneproject.content_rating.spelling_correction.spellchecker module¶

SpellChecker Module; simple, intuitive spell checker based on the post by Peter Norvig. See: https://norvig.com/spell-correct.html Code from https://github.com/barrust/pyspellchecker

class capstoneproject.content_rating.spelling_correction.spellchecker.SpellChecker(language='en', local_dictionary=None)[source]¶

Bases: object

The SpellChecker class encapsulates the basics needed to accomplish a simple spell checking algorithm. It is based on the work by Peter Norvig (https://norvig.com/spell-correct.html)

Parameters:
  • language (str) – The language of the dictionary to load or None for no dictionary. Supported languages are en, es, de, and fr. Defaults to en
  • local_dictionary (str) – The path to a locally stored word frequency dictionary
